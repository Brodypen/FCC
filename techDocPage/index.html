List items

Using lists allows developers to group like-items together

Unordered lists looks like this

  • Item in a unordered list!
  • This one too
  • Wow very unordered!

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

Note: Unordered list by default includes bullet points

Order lists looks like this

  1. Item in a ordered list!
  2. Looks like steps!
  3. Wow very ordered!

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

Note: Ordered list by default includes numbers
